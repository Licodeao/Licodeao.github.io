# 关于a标签

  1. 外部链接：target属性的默认值是_self 表示当前窗口打开，_blank表示新窗口打开.
  2. 内部链接：只需要填入HTML文件名即可
  3. 空链接：填入# （href = "#"）
  4. 下载链接：填入zip等压缩包名即可

# 关于form标签

form标签会把它范围内的表单元素信息提交给服务器，action属性值应为url地址，method="get/post" 设置表单数据的提交方式，name="表单名称", type="file" 表示文件域

label标签为input元素定义标注，用于绑定一个表单元素，当点击label标签内的文本时，浏览器就会自动将焦点（光标）转到或者选择对应的表单元素上，用来增加用户体验。<label for = "id属性"></label>标签的for属性应当与相关元素的id属性相同。

HTML5新增了许多input表单，但验证时需要在form表单域内。

新增的表单属性：

​ require -> 表示其内容不能为空，必填

​ placeholder -> 表单的提示信息

​ autofocus -> 页面加载完成自动聚焦到指定表单

​ autocomplete -> 显示之前填过的值

​ multiple -> 可以多选文件提交

# 语义化标签：

  1. header 头部标签

  2. nav 导航标签

  3. article 内容标签

  4. section 定义文档某个区域 (类似div)

  5. aside 侧边栏标签

  6. footer 尾部标签

    在IE9中，需要把这些标签转换成块级元素，更多地使用在移动端。

# 多媒体标签：

​ 视频标签：

​ controls -> 播放控件

​ autoplay属性：视频就绪自动播放(chrome需要添加muted属性来解决不能自动播放视频的问题)

​ 音频标签：

​ controls -> 播放控件

​ chrome依然把音频自动播放禁止了
